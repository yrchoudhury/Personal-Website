Task 3 / Typography Posters

Here is Task 3's Typography Poster we have made which reflect on the 8 different types of Typography.

Axial System - Axial Designs are usually symmeterical or asymmetrical. Their elements mostly organise to the left or right of the axis.This is because of the arrangement of the implied main line.

Bilateral System - The bilateral system is the most proportioned of the visual organization systems. It comprises of a one axis with lines of text placed on the axis.

Radial System - Usually in the radial system all text / elements are sorted to extend from the center point of focus like a line.In this system, the text are arranged to be read in loads of different ways. Such as: top to bottom, bottom to top and so forth, still making a message.

Dilatational System - In the dilatational system it's usually circles in the system which expand from the central point.The simplest forms of the dilatational system are circles that increase in regular or flowing increases from the centre.

Random System - The random system contain sections that are positioned with no distinct intention, design, path, rule, technique or intent, but it is deceivingly straightforward because the viewer requires organization on structures even when it is unintentional.

Grid System - A grid is a structure of vertical and horizontal branches that organize and create connections between components. Grid system arrangements are generally formal and are expected to produce visible order and economy in construction.

Modular System - The modular system is reliant on identical non-objective aspects or parts that act as a base to hold and include text. Compositions are created by the organization and positioning of the modular units.
